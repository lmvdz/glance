glance takes no config file — everything is environment variables plus a small on-disk state directory. Every variable and default below is traced to its read site in `src/`. Variables with no default are simply unset (the feature stays off or falls back as noted). See [`.env.example`](https://github.com/lmvdz/glance/blob/main/.env.example) for a paste-ready template.

<Callout type="info">
**Naming.** The canonical prefix is **`GLANCE_*`**; the legacy **`OMP_SQUAD_*`** spelling of every prefixed variable is honored indefinitely (`src/env-compat.ts` mirrors the two at boot; `GLANCE_*` wins when both are set). Unprefixed vars (`PLANE_*`, `DATABASE_URL`, `BETTER_AUTH_*`, `GITHUB_*`, `WORKOS_*`, `OMP_EXT_DIR`) have a single spelling.
</Callout>

## Enums

### AgentStatus

The derived, human-meaningful lifecycle state of one managed agent.

| Value | Meaning |
| --- | --- |
| `starting` | Process spawned, awaiting the RPC `ready` frame. |
| `working` | An agent turn is actively streaming. |
| `idle` | Ready, turn finished, awaiting the next instruction. |
| `input` | Blocked on a human decision (approval / question / tool input). |
| `error` | Spawn failed, child crashed, or fatal RPC error. |
| `stopped` | Intentionally terminated. |

### AgentKind

Which runtime backs a managed agent.

| Value | Meaning |
| --- | --- |
| `omp-operator` | A standard `omp --mode rpc` child in a worktree. |
| `flue-service` | A commissioned worker invoked through its Flue workflow. |
| `workflow` | A `.fabro` workflow graph (or a synthesized verify loop) driving the run. |

### ApprovalMode

| Value | Meaning |
| --- | --- |
| `always-ask` | Confirm every tool action. |
| `write` | Auto-approve reads/writes; the default for fleet agents. |
| `yolo` | Auto-approve everything. |

### ThinkingLevel

Reasoning effort, ordered low → high: `minimal`, `low`, `medium`, `high`, `xhigh`. Fleet agents default to `low` so they stay responsive.

## SquadEvent

The manager → surface event union, broadcast over the WebSocket. See the [WS protocol](/docs/reference/http-api).

```ts title="src/types.ts"
type SquadEvent =
	| { type: "roster"; agents: AgentDTO[]; version: string }
	| { type: "agent"; agent: AgentDTO }
	| { type: "removed"; id: string }
	| { type: "transcript"; id: string; entry: TranscriptEntry }
	| { type: "log"; level: "info" | "warn" | "error"; text: string }
	| { type: "commands"; id: string; commands: CommandInfo[] }
	| { type: "features-changed" };
```

| Variant | Meaning |
| --- | --- |
| `roster` | Full roster snapshot; `version` is the served-UI fingerprint. |
| `agent` | One agent's DTO changed. |
| `removed` | An agent left the roster. |
| `transcript` | One new transcript entry for an agent. |
| `log` | A daemon log line. |
| `commands` | The slash commands available to an agent changed. |
| `features-changed` | The feature set changed; refetch `/api/features`. |

## ClientCommand

The surface → manager command union, posted to `/api/command` or sent over the WebSocket.

```ts title="src/types.ts"
type ClientCommand =
	| { type: "prompt"; id: string; message: string }
	| { type: "answer"; id: string; requestId: string; value: string }
	| { type: "interrupt"; id: string }
	| { type: "kill"; id: string }
	| { type: "restart"; id: string }
	| { type: "remove"; id: string; deleteWorktree?: boolean }
	| { type: "create"; options: CreateAgentOptions }
	| { type: "snapshot" }
	| { type: "subscribe"; id: string }
	| { type: "commission"; spec: CommissionSpec };
```

| Variant | Meaning |
| --- | --- |
| `prompt` | Send an instruction to an agent. |
| `answer` | Answer a `PendingRequest` by its `requestId`. |
| `interrupt` | Interrupt the current turn. |
| `kill` | Terminate the agent process. |
| `restart` | Restart the agent. |
| `remove` | Remove the agent; optionally delete its worktree. |
| `create` | Spawn a new agent from `CreateAgentOptions`. |
| `snapshot` | Request a full roster + recent transcript replay. |
| `subscribe` | Ask for a transcript replay of one agent (unicast). |
| `commission` | Run the commissioning loop for a `CommissionSpec`. |
